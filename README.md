# Satvika — Nutrition Label Generator

> KIH Hackathon Project

Satvika is an AI-powered nutrition label generator built to help food businesses comply with FSSAI regulations. It automates ingredient analysis, allergen detection, nutritional calculations, and generates print-ready labels in PDF format.

---

## Tech Stack

**Backend:** Python · Django 5 · Django REST Framework · Simple JWT · SQLite · ReportLab · Pandas  
**Frontend:** React 18 · Vite 6 · React Router · Axios · Recharts · GSAP · Anime.js
